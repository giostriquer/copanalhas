# Contributing

Copanalhas is public source, but it is primarily maintained for one private
Discord server and is not currently accepting external pull requests.

If you find something useful or spot a safety issue, opening an issue is welcome.
For feature work, please fork the repository for your own server instead of
opening a PR against this one.

Do not share Discord bot tokens, Football-Data API tokens, guild IDs, channel
IDs, runtime SQLite files, logs, screenshots with secrets, or private Discord
message content in issues.
